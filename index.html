Projetos de Ciência de Dados

Previsão de vendas das farmácias Rossmann

A partir de um conjunto de dados da Rossmann, pude analisar e então construir um modelo de Machine Learning que pudesse prever as vendas nas próximas 6 semanas. Em seguida, construí uma API que calculasse a previsão para o conjunto de dados inserido com o modelo treinado. Publiquei a API no Heroku e a vinculei a um bot no Telegram, que iria responder a previsão para um determinado id de farmácia, depois de solicitada. Saiba mais.

Health Insurance Cross Sell Prediction

Cross Sell é uma estratégia para tentar vender um produto para um cliente que já adquiriu outro produto do mesmo vendedor. Nesse projeto, o objetivo é prever quais clientes de uma rede de seguros de saúde estariam interessados em um plano de seguro de carros também. Implementei uma solução utilizando Learning to Rank, em que um modelo de Machine Learning preveu uma pontuação para cada pessoa e rankeou o conjunto de dados, indicando os clientes mais suscetíveis a aderir ao outro seguro. Construí uma API que calcula o a pontuação de cada cliente para um conjunto de dados inserido e publiquei no Heroku. Vinculei a API ao Google Sheets em que é mostrada a previsão para cada registro inserido. Saiba mais.

Construção de um modelo de manutenção preditiva para falha de turbofans de aviões

Construí um modelo de predição de falha de turbofans para identificar se estariam próximas ou não de uma falha completa. Ao final, o modelo apresentou 80% de precisão na previsão da falha das turbofans. Utilizei python, matplotlib e seaborn para as análises e LightGBM para treinar o modelo. Saiba mais.
